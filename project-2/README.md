
# Revenue Forecasting and Performance Analysis of Vodafone Qatar (2021–2023)

## Project Overview

This project transformed Vodafone Qatar’s telecom data into an enriched, ML-ready dataset using NexusMax, Python, and Microsoft Fabric. It enabled advanced revenue forecasting, churn analysis, and performance tracking through a comprehensive Power BI dashboard. The solution empowered finance and strategy teams with actionable insights for data-driven decision-making.

---

## Business Problem

Vodafone Qatar faced challenges in understanding revenue fluctuations and churn across regions and customer segments due to fragmented data. This lack of consolidated insights hindered accurate forecasting and strategic budget planning. Finance and strategy teams required a unified view of key metrics to reduce uncertainty and improve decision confidence.

---

## Objective

- Create a unified, clean, and ML-ready telecom dataset for analysis
- Improve quarterly revenue forecast accuracy to optimize investments
- Reduce churn in key customer segments to protect recurring revenue

---

## Tools & Technologies

- NexusMax
- Python
- Microsoft Fabric
- Power BI
- NumPy
- Pandas

---

## Project Workflow

- Extract and clean raw Vodafone Qatar telecom data
- Engineer features and create a centralized Date table for time analysis
- Apply supervised regression models for quarterly revenue forecasting
- Develop Power BI dashboards to visualize revenue, churn, and network KPIs
- Deliver unified ML-ready dataset and insights to finance and strategy teams

---

## Key Insights

- Total revenue increased 14% quarter-over-quarter, driven primarily by Doha region
- Churn rate for high-value prepaid customers rose by 0.3% QoQ, impacting revenue stability
- Network availability remained high at 99.95%, supporting strong customer retention efforts
- Churn in Mobile Postpaid and Fixed Broadband segments exceeded overall churn, risking $8–12K annual revenue loss
