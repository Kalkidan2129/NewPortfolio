
# Global Superstore Sales Dashboard

## Project Overview

This project created a comprehensive sales dashboard for a global superstore using Power BI to visualize sales data by country, segment, category, and subcategory. It enabled stakeholders to analyze sales trends and performance during the pandemic-driven shift to online shopping. The dashboard supports data-driven decision-making by integrating multiple data points into clear visualizations.

---

## Business Problem

The global pandemic drastically changed consumer shopping behavior, increasing online purchases and complicating sales analysis across regions and product categories. Retailers needed a unified view to understand these shifts and optimize sales strategies. Without consolidated insights, identifying growth opportunities and addressing challenges was difficult.

---

## Objective

- Build a dynamic dashboard to track sales performance by geography and product segments
- Enable visualization of sales trends to support strategic decision-making
- Provide actionable insights to improve sales and customer targeting

---

## Tools & Technologies

- Power BI
- DAX
- Excel
- Power Query
- Data Modeling
- Data Visualization

---

## Project Workflow

- Collect and prepare sales data from multiple sources
- Build data model and relationships in Power BI
- Create KPIs and calculated measures using DAX
- Design and develop interactive visualizations and slicers
- Publish and refine dashboard for stakeholder use

---

## Key Insights

- Sales increased significantly in online channels during the pandemic period
- Certain product categories outperformed others across different countries
- Segment analysis revealed shifting customer preferences during lockdowns
- Monthly sales trends highlighted seasonal and pandemic-related fluctuations
