
# 30-Day Hospital Readmission Insights and Cost Trends

## Project Overview

This project leverages machine learning techniques, including Logistic Regression and Random Forest, to predict patients at high risk of 30-day hospital readmission. It integrates Python, Microsoft Fabric, and Power BI to transform raw hospital data into actionable clinical insights via an executive dashboard. Stakeholders can explore readmission patterns, cost drivers, and risk segments to enable early intervention and improve patient outcomes.

---

## Business Problem

Unplanned 30-day hospital readmissions represent a significant financial and clinical burden for healthcare providers. Identifying patients at risk before discharge remains a challenge, limiting opportunities for timely intervention. This project addresses the need for predictive tools to reduce readmission rates and associated costs.

---

## Objective

- Predict high-risk patient readmissions within 30 days using clinical and demographic data
- Develop an interactive dashboard for stakeholders to explore risk factors and cost trends
- Enable data-driven early interventions to reduce readmission rates and healthcare costs

---

## Tools & Technologies

- Python
- Microsoft Fabric
- Power BI
- Jupyter Notebook
- SQL Server
- Excel
- Microsoft Azure

---

## Project Workflow

- Import and clean hospital admission data
- Remove duplicates and filter placeholder values
- Build data relationships and create date and KPI tables
- Train and compare Logistic Regression and Random Forest models
- Deploy interactive Power BI dashboard for executive insights

---

## Key Insights

- Certain patient demographics and clinical profiles strongly predict readmission risk
- Readmission rates correlate with specific cost drivers and patient volume trends
- Risk segmentation enables targeted interventions for high-risk groups
- Dynamic dashboards improve stakeholder understanding of readmission patterns
