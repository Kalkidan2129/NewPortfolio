
# Conflict in Ethiopia (1997 - 2015)

## Project Overview

This project analyzed recorded conflicts in Ethiopia from 1997 to 2015, focusing on ethnic tensions and resource disputes. Using Power BI, it visualized deaths and conflict trends by interaction groups to identify key conflict drivers. The analysis highlighted how federal ethnic-based structures contribute to ongoing disputes.

---

## Business Problem

Ethiopia faces persistent ethnic conflicts driven by disputes over land, resources, and political power, resulting in injuries, deaths, and displacement. Understanding these conflicts is critical for policymakers to address root causes and improve regional stability. However, fragmented data and complex ethnic dynamics hinder effective conflict resolution.

---

## Objective

- Visualize conflict data to identify patterns and trends over time
- Analyze causes of ethnic conflicts related to land and resource disputes
- Provide actionable insights to support government policy and conflict mitigation

---

## Tools & Technologies

- Power BI
- Data Visualization
- Human Rights Council Data
- Line and Stacked Column Charts
- Data Analytics Techniques

---

## Project Workflow

- Collect and preprocess conflict data from Human Rights Council records
- Identify key variables such as deaths, interaction groups, and time periods
- Create line and stacked column charts in Power BI to visualize deaths and % changes
- Format visuals for clarity including legends, axes, and data labels
- Filter data by interaction groups to analyze specific conflict drivers

---

## Key Insights

- Most conflicts stem from disputes over land and geographical boundaries between ethnic groups
- Federal ethnic-based governance structure exacerbates tensions and disparities
- Access to state resources and language policies are significant conflict triggers
- Conflict trends show fluctuations in deaths with notable year-over-year and month-over-month changes
