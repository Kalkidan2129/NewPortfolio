
# Revenue Forecasting and Performance Analysis of Vodafone Qatar (2021–2023)

## Project Overview

This project transformed Vodafone Qatar's telecom data using NexusMax, Python, and Microsoft Fabric to enable advanced revenue forecasting and performance analysis. It uncovered patterns in revenue, churn, and customer behavior, supporting data-driven decision-making for finance and strategy teams. The final deliverable included a comprehensive Power BI dashboard visualizing key KPIs from 2021 to 2023.

---

## Business Problem

Telecom revenue performance varied significantly across regions and customer segments, with no consolidated view to identify growth or risk areas. Fluctuations in churn and subscriber behavior created uncertainty in forecasting and budget planning. Finance and strategy teams lacked confidence in data-driven decisions without a unified view of revenue, subscribers, churn, and network performance over time.

---

## Objective

- Create a unified, clean, and ML-ready dataset from raw telecom data
- Improve quarterly revenue forecasting accuracy to optimize budget allocation
- Reduce churn in key customer segments to protect recurring revenue

---

## Tools & Technologies

- NexusMax
- Python
- Microsoft Fabric
- Power BI
- NumPy
- Pandas

---

## Project Workflow

- Data extraction and cleaning from Vodafone Qatar datasets
- Data transformation and feature engineering using Python and NexusMax
- Creation of a centralized date table for time-based analysis
- Development of supervised regression models for revenue forecasting
- Visualization of key KPIs and trends in Power BI dashboards

---

## Key Insights

- Total revenue increased 14% quarter-over-quarter, driven largely by Doha's 72M subscribers
- Churn rate for high-value prepaid customers rose by 0.3% QoQ, impacting revenue stability
- Network availability remained high at 99.95%, supporting strong customer experience
- Churn in Mobile Postpaid and Fixed Broadband segments exceeded overall churn, causing estimated annual revenue loss of $8–12K
