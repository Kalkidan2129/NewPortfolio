
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created a comprehensive Power BI dashboard to analyze the effects of pilot awareness on wildlife strike incidents. It compares outcomes between well-informed warned pilots and uninformed pilots to assess damage costs and passenger safety. The data was sourced and modeled using SQL Server to enable detailed reporting and insights.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and can lead to costly damages and passenger injuries. There is a need to understand how pilot awareness and timely warnings influence these outcomes. Improving pilot information could reduce damage costs and enhance flight safety.

---

## Objective

- Analyze the impact of pilot warnings on wildlife strike damage costs
- Compare safety outcomes between informed and uninformed pilots
- Develop an interactive dashboard for data-driven decision making

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL scripting
- Data modeling
- Dashboard reporting

---

## Project Workflow

- Load and prepare wildlife strike data using SQL Server
- Import and model data within Power BI
- Create visual elements including slicers and KPI cards
- Develop measures to compare warned vs. unwarned pilot impacts
- Design and finalize the interactive dashboard for insights

---

## Key Insights

- Well-informed warned pilots contribute to lower damage costs from wildlife strikes
- Passenger safety improves when pilots receive timely wildlife strike warnings
- Data slicing by time and flight phase reveals patterns in strike incidents
- Interactive KPIs highlight critical metrics such as total deaths and injuries
