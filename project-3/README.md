
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created a comprehensive Power BI dashboard to analyze the effects of pilot warnings on wildlife strike damage and passenger safety. It involved data extraction from SQL Server, data modeling, and KPI creation to provide actionable insights. The dashboard enables comparison between informed and uninformed pilots to control repair costs and enhance flight safety.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and cause costly damages. There is a need to understand how pilot awareness and timely warnings influence damage control and passenger safety during flights. Improving this understanding can help reduce repair costs and enhance operational safety.

---

## Objective

- Analyze impact of pilot warnings on wildlife strike damage and safety
- Develop interactive dashboard for data visualization and insights
- Create KPIs to monitor warning effectiveness and damage trends

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL scripting
- Data modeling

---

## Project Workflow

- Load and prepare wildlife strike data using SQL Server
- Import and model data within Power BI
- Create slicers for time and flight phase filtering
- Develop KPI cards to measure pilot warning impact
- Design and finalize dashboard with logos and titles

---

## Key Insights

- Well-informed pilots reduce wildlife strike damage costs significantly
- Timely warnings correlate with fewer injuries and fatalities
- Year-over-year analysis shows trends in pilot warning effectiveness
- Data slicers enable detailed examination by time period and flight phase
