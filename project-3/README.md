
# US SCHOOL SHOOTING DASHBOARD

## Project Overview

This project created a comprehensive Power BI dashboard analyzing school shooting fatalities and trends across US states, school types, and categories over 30 years. It involved data cleaning, transformation, calendar table creation, and measure calculations to enable insightful visualizations. The dashboard supports decision-making to enhance school security systems nationwide.

---

## Business Problem

School shootings have become a recurring and devastating issue in the United States, impacting educational environments and safety. There is a critical need to understand patterns and trends in these incidents to inform effective security measures and policy decisions.

---

## Objective

- Analyze year-over-year and monthly trends in school shooting fatalities
- Segment data by school type, category, and state for detailed insights
- Develop an interactive dashboard to support data-driven security enhancements

---

## Tools & Technologies

- Power BI
- Excel
- DAX
- Data Transformation
- Data Modeling
- Data Warehousing
- ETL Processes

---

## Project Workflow

- Import and load Excel data into Power BI
- Clean and transform data, replacing nulls and formatting columns
- Create calendar table and establish relationships
- Develop calculated columns and DAX measures for analysis
- Design and deploy interactive dashboard for insights

---

## Key Insights

- Fatalities analyzed by month and year reveal temporal shooting trends
- Different school types show varying patterns in shooting incidents
- Geographical distribution highlights states with higher incident rates
- Categorization of shootings provides deeper understanding of incident nature
