
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created a comprehensive Power BI dashboard to analyze the effect of pilot warnings on wildlife strike damage and flight safety. It involved data extraction from SQL Server, data modeling, KPI creation, and interactive visualizations. The dashboard highlights the differences between informed and uninformed pilots to support decision-making in aviation safety.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and cause costly damages. There is a need to understand how pilot awareness and timely warnings can reduce these impacts and improve passenger safety. Without actionable insights, airlines face increased repair costs and potential safety hazards.

---

## Objective

- Analyze the impact of pilot warnings on wildlife strike damage costs
- Create interactive visualizations to monitor key safety metrics
- Enable data-driven decisions to enhance flight safety and reduce repair costs

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL scripting
- Data modeling

---

## Project Workflow

- Load and prepare wildlife strike data using SQL scripts
- Import and model data within Power BI
- Create slicers and interactive filters for time and flight phases
- Develop KPI cards to track pilot warning percentages and damage metrics
- Design and format dashboard with logos, maps, and legends for clarity

---

## Key Insights

- Well-informed pilots significantly reduce wildlife strike damage costs
- Regions with higher warned pilot percentages show improved safety outcomes
- Year-over-year trends indicate changes in pilot awareness and damage incidence
- Interactive slicers enable detailed analysis by time period and flight phase
