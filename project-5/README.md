
# US SCHOOL SHOOTING DASHBOARD

## Project Overview

This project involved creating a comprehensive Power BI dashboard to analyze school shooting incidents across the United States by month, category, school type, and state from 1990 to 2020. The analysis focused on percentage fatalities and year-over-year trends to provide actionable insights. The dashboard enables stakeholders to better understand the patterns and severity of school shootings to inform security decisions.

---

## Business Problem

School shootings have become a recurring and critical issue in the US, causing loss of life and raising urgent security concerns in educational institutions. There is a lack of consolidated data analysis to understand trends by time, location, and school type. Without this insight, it is challenging to implement effective preventative measures and enhance school safety protocols.

---

## Objective

- Analyze percentage fatalities and trends of school shootings from 1990 to 2020
- Segment data by month, category, school type, and state for detailed insights
- Develop an interactive dashboard to support decision-making on school security

---

## Tools & Technologies

- Power BI
- Excel
- DAX
- Data Transformation (ETL)
- Data Modeling
- Data Visualization
- Calendar Table Creation

---

## Project Workflow

- Import and load Excel data into Power BI
- Transform and clean data, replacing null values appropriately
- Create calendar table and establish relationships
- Develop calculated measures using DAX for fatalities and trends
- Design and deploy interactive dashboard with slicers and visuals

---

## Key Insights

- School shootings show varying fatality percentages by month and year, indicating temporal trends
- Certain school types and categories experience higher incident rates and fatalities
- Geographic analysis reveals state-level differences in school shooting occurrences
- Year-over-year data highlights periods of increased or decreased violence, informing security focus
